## DOL v0.3.0 "Pragmatic Evolution"

### New Features
- **HIR Architecture**: 22 canonical node types for cleaner codegen
- **`type` keyword**: Alias for `gene` (v0.3.0 syntax)
- **`extends` clause**: `gene Foo extends Bar { ... }`
- **`val`/`var` bindings**: Alternative to `let`/`mut`
- **Inline exegesis**: Exegesis can now appear inside gene blocks

### Fixes
- Parser now correctly handles inline exegesis
- Empty exegesis is a warning, not an error

### Breaking Changes
- None (backward compatible with v0.2.3)
