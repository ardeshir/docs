### New Features

#### ENR Bridge Module (`enr_bridge/`)

- **GradientBroadcaster**: Propagates resource availability (CPU, memory, storage) via gossipsub
- **CreditSynchronizer**: Handles credit transfers with 2% entropy tax
- **DistributedElection**: Nexus election protocol for hub node selection
- **SeptalGateManager**: Circuit breakers for isolating unhealthy nodes

#### Docker Integration Testing

- **Dockerfile.integration**: Multi-stage build with all univrs-* dependencies
- **docker-compose.test.yml**: Container orchestration with cargo caching
- **TestCluster helper**: Spawns multi-node networks for integration tests
