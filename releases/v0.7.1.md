# DOL v0.7.1 Release Notes

**Release Date:** January 2025
**Codename:** Polish

---

## Overview

DOL v0.7.1 is a patch release focused on documentation improvements, example files, and preparing for crates.io publication.

---

## Changes

### New Example Files

Added example DOL files demonstrating language features:

- Basic gene definitions
- Trait implementations
- Effect system usage
- WASM compilation examples

### Documentation Updates

- Removed "production-ready" claim from README (honest about project status)
- Expanded exclude list for cleaner crate publish
- Updated .gitignore for cargo publish

### CI/CD Fixes

- Removed redundant WASM test step that was hanging on Ubuntu runners

---

## Installation

```bash
# From crates.io
cargo install metadol --version 0.7.1

# From source
git checkout v0.7.1
cargo build --release
```
